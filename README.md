# HitMarkers for S.T.A.L.K.E.R. 2

## What It Does

HitMarkers adds a small on-screen marker when the player damages enemies in S.T.A.L.K.E.R. 2:

- White hit marker when the player hits an enemy.
- Red hit marker when the player kills an enemy.

## Installation

1. Download the latest HitMarkers release package.
2. Open your S.T.A.L.K.E.R. 2 installation folder.
3. Go to `Stalker2\Content\Paks`.
4. Create a folder named `~mods` if it does not already exist.
5. Copy the HitMarkers package files into `Stalker2\Content\Paks\~mods`.
6. Launch the game.
